# Epic: Veritas Micro-MVP
**ID:** EPIC-VERITAS-001
**Title:** Decentralized Stake-Backed Journalism Micro-MVP
**Status:** TODO
**Level:** 1

## Description
Implementation of the core Veritas Protocol lifecycle: Wallet-based identity, IPFS article storage, Arbitrum NFT provenance, and off-chain trust signatures.

## Success Criteria
- [ ] Users can connect via RainbowKit.
- [ ] Articles are stored on IPFS and represented by an ERC-721 on Arbitrum Sepolia.
- [ ] Users can "Upvote" articles via Web3 signatures stored in Supabase.
- [ ] A feed displays all minted articles with their community Trust Scores.

## Stories
- **STORY-VERITAS-01:** Web3 Foundation & Article Contract
- **STORY-VERITAS-02:** Markdown Editor & IPFS/NFT Minting Pipeline
- **STORY-VERITAS-03:** Credibility Feed & Signature-Based Trust Signals
